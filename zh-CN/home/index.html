Apache StreamPark™

Make stream processing easier

Apache StreamPark™ is a user-friendly streaming application development framework and one-stop cloud-native real-time computing platform.

Get StartGithub
https://streampark.apache.org
Start using Apache StreamPark™ in minutes
$curl -L https://streampark.apache.org/quickstart.sh | sh

Or install other release version from the download page.

Key Features

Development Framework

Provides a framework to simplify Apache Flink® and Apache Spark™ application development, with plug-and-play connectors that reduce the learning curve and development complexity, improving efficiency.

Real-time Computing Platform

Provides a one-stop streaming computation platform, with core functionalities including but not limited to application development, deployment, management, operation, and real-time data warehousing, etc.

Supports Batch & Streaming

Supports both Apache Flink® and Apache Spark™, enabling seamless integration of streaming and batch processing, with multi-engine/multi-version support on a single platform.

Rich Ecosystem

Integrates with a variety of open-source tools, including Apache Flink®, Apache Spark™, Apache Paimon™, Apache Doris, and ML/AI. Offering users greater flexibility in technology choices.

Easy to use

Apache StreamPark™ is designed to lower the learning curve and entry barrier. It requires only one service, making deployment easy and allowing even beginners to get started within minutes.

Mature and Stable

Widely deployed in large-scale production environments, offering second-level application state tracking, real-time monitoring, and alert notifications to ensure stability and support ongoing project growth.
